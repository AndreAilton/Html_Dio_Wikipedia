# HTML Tags Reference Index

Este repositório Tem como objetivo simular uma pagina do wikipedia com base nos conhecimentos adquiridos na aula, juntamente com links para a documentação na MDN Web Docs. Este índice foi criado para facilitar o aprendizado e a referência rápida das tags HTML mais comuns.

## Conteúdo

- [Formatação de Texto](#formatação-de-texto)
- [Listas](#listas)
- [Links e Imagens](#links-e-imagens)
- [Tabelas](#tabelas)
- [Formulários](#formulários)
- [Outros](#outros)

## Formatação de Texto

- `<p>` - Parágrafo
- `<h1>` to `<h6>` - Cabeçalhos
- `<strong>` - Negrito
- `<em>` - Itálico
- `<br>` - Quebra de linha
- `<hr>` - Linha horizontal
- `<blockquote>` - Citação em bloco

## Listas

- `<ul>` - Lista não ordenada
- `<ol>` - Lista ordenada
- `<li>` - Item de lista

## Links e Imagens

- `<a>` - Hiperlink
- `<img>` - Imagem

## Tabelas

- `<table>` - Tabela
- `<tr>` - Linha de tabela
- `<th>` - Cabeçalho de tabela
- `<td>` - Célula de tabela
- `<thead>` - Cabeçalho da tabela
- `<tbody>` - Corpo da tabela
- `<tfoot>` - Rodapé da tabela

## Formulários

- `<form>` - Formulário
- `<input>` - Campo de entrada
- `<label>` - Rótulo
- `<textarea>` - Área de texto
- `<button>` - Botão
- `<select>` - Lista suspensa
- `<option>` - Opção de lista suspensa

## Outros

- `<div>` - Divisão ou seção
- `<span>` - Seção de texto inline
- `<header>` - Cabeçalho de documento
- `<footer>` - Rodapé de documento
- `<nav>` - Links de navegação
- `<section>` - Seção de documento
- `<article>` - Conteúdo independente
- `<aside>` - Conteúdo adicional
- `<main>` - Conteúdo principal
- `<figure>` - Conteúdo ilustrativo
- `<figcaption>` - Legenda de figura

Cada tag HTML inclui um link para a documentação oficial da MDN Web Docs, onde você pode encontrar mais informações sobre seu uso e exemplos.
